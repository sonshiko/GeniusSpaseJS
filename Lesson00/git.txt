1. Увійти у власний акаунт https://github.com/ і створити там новий репозиторій.
2. На комп'ютері, на якому буде створюватись код, створити теку з потрібною назвою
   в редакторі коду VSCode або за допомогою інших інструменів (провідник, тотал коммандер, інше)
3. Зробити цю теку поточною.
   Далі виконати намтупні команди, наприклад у VSCode
4. git clone <адреса репозиторію> - ця команда синхронізує вашу току і ваш новий репозиторій на гіті
   Після цього можна працювати
5. git add <файл> - додасть файл в коміт, а git add . - додає всі зміни
6. git commit -m "текст"
7  git push
